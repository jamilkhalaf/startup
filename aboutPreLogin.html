About the Memory Game

Welcome to the Memory game, a modern twist on the nostalgic arcade classic! Our platform combines the beloved gameplay of Memory with modern web technologies to create an engaging experience.

Key features include:

  • Real-time leaderboard updates to track your elapsed time.
  • A responsive design for both desktop and mobile gameplay.
  • Random fun images from the Dog API after every game to celebrate your progress!

Log in or sign up to start playing and see how high you can climb on the leaderboard. Challenge yourself, have fun, and enjoy the blend of nostalgia and technology!
